# Contributing to EpicChain SDK

First off, thank you for considering contributing to the EpicChain SDK! It's people like you that make the open-source community such a great place.

## How to Contribute

There are many ways to contribute to the EpicChain SDK, from writing tutorials or blog posts, improving the documentation, submitting bug reports and feature requests or writing code which can be incorporated into the SDK itself.

### Reporting Bugs

If you find a bug, please open an issue on our [GitHub repository](https://github.com/epicchainlabs/epicscan-sdk-ts/issues). Please include as much detail as possible, including:

*   A clear and descriptive title.
*   A description of the bug, including the expected and actual behavior.
*   Steps to reproduce the bug.
*   The version of the SDK you are using.
*   Any relevant code snippets.

### Suggesting Enhancements

If you have an idea for a new feature or an improvement to an existing one, please open an issue on our [GitHub repository](https://github.com/epicchainlabs/epicscan-sdk-ts/issues). Please include:

*   A clear and descriptive title.
*   A detailed description of the enhancement you are proposing.
*   Any relevant code snippets or mockups.

### Pull Requests

If you would like to contribute code to the SDK, please open a pull request on our [GitHub repository](https://github.com/epicchainlabs/epicscan-sdk-ts/pulls). Please ensure that your pull request:

*   Follows the existing code style.
*   Includes tests for any new or changed functionality.
*   Is well-documented.

## Development Setup

To get started with development, you will need to have Node.js and pnpm installed.

1.  Clone the repository: `git clone https://github.com/epicchainlabs/epicscan-sdk-ts.git`
2.  Install dependencies: `pnpm install`
3.  Run the tests: `pnpm test`

## Code of Conduct

Please note that this project is released with a [Contributor Code of Conduct](./CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
